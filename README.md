
# Orange-HRM end-to-end cypress test

Create Cypress tests to cover the workflow of Orange-HRM application for comprehensive end-to-end test coverage.

## Deployment

To deploy this project run need npm and node installed.

```bash
  npm install
```
```bash
  npx cypress open
```
